# matrix

This is a React Three Fiber project that recreates the digital rain effect from The Matrix.

It runs as a Vite app using React and TypeScript, with the scene rendered through `@react-three/fiber` and Three.js.

The active app entrypoint is wired to the code in `src/text-rain`, which contains the Matrix digital rain implementation currently used by the project.

Live demo: https://s4njee.github.io/matrix-3js/

## Run the dev server

1. Install dependencies:

```bash
npm install
```

2. Start the Vite dev server:

```bash
npm run dev
```

3. Open the local URL printed in the terminal, which is typically:

```text
http://localhost:5173
```
